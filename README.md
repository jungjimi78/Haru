# 오늘

하루를 시간 아카이브로 기록하는 작은 PWA 앱.

## 기능
- 05:00~23:00 하루 시간 아카이브
- 날짜 이동 / 달력 선택
- 지난 기록 보기, 미래 날짜 계획 입력
- 블록별 기록: 묵상, 오늘의 표현, 독서, 한끼, 감사 등
- 하단 메뉴: 최근 7일 모아보기
- 데이터는 브라우저 localStorage에 저장

## GitHub Pages 배포
1. 이 폴더를 GitHub 저장소에 업로드
2. Settings → Pages
3. Branch를 `main`, folder를 `/root`로 선택
4. 배포 주소에서 `index.html` 열기

## iPhone 홈 화면 추가
Safari에서 배포 주소 열기 → 공유 버튼 → “홈 화면에 추가”
